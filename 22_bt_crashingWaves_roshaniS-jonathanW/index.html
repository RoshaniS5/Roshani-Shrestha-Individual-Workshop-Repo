1. Navbar

Documentation about the navbar. The overall navbar is spanned by a navbar class. The background color (navy) is set by adding it in the style attribute in the nav tag. The color scheme of dark is set by adding "navbar-dark" in the class attribute of the nav tag. If you make the window smaller, "collapse navbar-collapse" would collapse the navbar into a hamburger style. In this case, we commented that out because it would most likely require JavaScript to work properly. Using "container-fluid" allows the navbar to span the entire width. Using "navbar-nav me-auto mb-2 mb-lg-0" styles the navbar so that all the items are in a row. This is used in the class attribute of the ul tag. This is also used to align all the items to the left and resizes the navbar properly. "nav-item" is used for adding items to the navbar. "nav-link" is used to add links to the navbar in the same style as the rest of the navbar. "active" and "disabled" can be used to specify the properties of these links. "d-flex" in the form tag organizes the elements within it (in this case, the search bar and button). This feature of prebuilt navbars in Bootstrap is a very handy feature for every website that needs a navigation bar.
